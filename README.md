# 2025년 육아휴직수당 계산기

공무원을 위한 정확한 육아휴직수당 계산 도구입니다. 2025년 정부 공식 기준(인사혁신처예규 제204호)에 따라 월별 지급액을 자동으로 계산합니다.

## 🎯 주요 기능

- **정확한 계산**: 2025년 공무원보수 등의 업무지침 기준 준수
- **자녀 순서 선택**: 첫째, 둘째, 셋째 이후 구분
- **휴직 기간 설정**: 시작일과 종료일 입력
- **월봉급액 입력**: 자동 포맷팅 지원
- **특수 조건 적용**:
  - 부부동시 육아휴직 (첫 3개월 50% 감액)
  - 한부모가족 (지급기간 18개월 연장, 1~3개월 상한액 300만원)
  - 장애아부모 (지급기간 18개월 연장)
- **월별 상세 내역**: 일할 계산, 감액 내용 명시
- **실시간 계산**: 입력값 변경 시 즉시 반영
- **모바일 반응형**: 모든 기기에서 최적화된 UI

## 🚀 시작하기

### 필수 요구사항

- Node.js 22.13.0 이상
- pnpm 10.4.1 이상

### 설치

```bash
# 프로젝트 클론
git clone https://github.com/yourusername/parental-leave-calculator.git
cd parental-leave-calculator

# 의존성 설치
pnpm install
```

### 개발 서버 실행

```bash
pnpm dev
```

브라우저에서 `http://localhost:3000`으로 접속하세요.

### 빌드

```bash
pnpm build
```

### 프로덕션 실행

```bash
pnpm start
```

## 📋 계산 규칙

### 기본 지급액 상한

| 기간 | 상한액 |
|------|--------|
| 1~3개월 | 250만원 |
| 4~6개월 | 200만원 |
| 7개월 이상 | 160만원 |

### 특수 조건별 규칙

#### 부부동시 육아휴직
- 같은 자녀에 대해 배우자가 먼저 육아휴직을 한 경우
- 첫 3개월 동안 지급액의 50% 감액 적용

#### 한부모가족
- 한부모가족지원법 제4조제1호 해당자
- 지급기간 18개월로 연장
- 1~3개월 상한액을 300만원으로 인상

#### 장애아부모
- 장애의 정도가 심한 자녀를 양육하는 경우
- 지급기간 18개월로 연장
- 장애인복지법 시행규칙 기준

### 계산 방식

- **지급액 계산**: 월봉급액과 상한액 중 낮은 금액
- **월중 시작/종료**: 일할 계산 적용 (30일 기준)
- **반올림**: 10원 단위 절사
- **최소 지급액**: 700,000원 (전월 기준)
